# Seoul_IoT_Hackathon 2018 - 화재알림서비스 (a.k.a 삐뽀)

## 서비스 소개
해당 서비스는 화재발생시 건물내의 비콘을 이용하여, 빠른 대피경로를 안내해줍니다. 또한 소방관, 안전관리자에게는 건물 내 인원의 위치를 파악할 수 있게 하여 효과적인 구출루트를 계획하는 것이 가능합니다.

## 역할배분
개발자3(프론트엔드-안드로이드앱, 백엔드, 임베디드)

디자이너2(UI)
